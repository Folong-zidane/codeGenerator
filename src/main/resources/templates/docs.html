API REST

POST /api/generate/crud

Genere un projet complet avec scripts de developpement continu

Request Body:
{
  "umlContent": "classDiagram\\n    class User {\\n        +UUID id\\n        +String username\\n    }",
  "packageName": "com.example",
  "language": "java"
}
Response:

Fichier ZIP telechargeable avec projet complet

POST /api/generate/validate

Valide la syntaxe UML

Request Body (text/plain):
classDiagram
    class User {
        +UUID id
        +String username
    }

Exemples Pratiques

Generation Java Spring Boot

curl -X POST https://codegenerator-cpyh.onrender.com/api/generate/crud \
  -H "Content-Type: application/json" \
  -d '{
    "umlContent": "classDiagram\\n    class User {\\n        +UUID id\\n        +String username\\n    }",
    "language": "java",
    "packageName": "com.example"
  }' \
  -o project.zip

Generation Python Django

curl -X POST https://codegenerator-cpyh.onrender.com/api/generate/crud \
  -H "Content-Type: application/json" \
  -d '{
    "umlContent": "classDiagram\\n    class User {\\n        +UUID id\\n        +String username\\n    }",
    "language": "django",
    "packageName": "com.blog"
  }' \
  -o blog.zip
